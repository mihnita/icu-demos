The current folder contains a JSP application. 
Each page is a demonstration of some i18n functionality implemented using ICU4J. 
Add an extra page that demonstrate collation, using `com.ibm.icu.text.Collator`. 
The collator should be configured with strength, normalization, numeric, and other flags, all exposed in the web UI. 
Use this page as a model: https://icu4c-demos.unicode.org/icu-bin/collation.html 
Do not touch anything else, only what is needed to do the job. 
If you get stuck, don't guess, ask.
